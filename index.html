Upload Folder Demo

Source code: https://github.com/komputronika/UploadFolder

Pure Javascript and pure PHP without any library

This page allows you to pick a folder via a HTML file input. All files (including nested files) inside that folder are added to the input's files list. This attribute is supported in Chrome, Edge, and Firefox.

Choose a folder, and all files in that folder will be uploaded recursivelty.
